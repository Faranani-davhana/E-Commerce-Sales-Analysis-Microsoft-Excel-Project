# Sales Performance Analysis 

## Project Overview

This project analyzes the sales data using Microsoft Excel to identify trends, evaluate product performance, and gain useful insights that will help to improve business decisions.

## Business questions and objectives

1. Analyze sales performance
2. Which product(s) generates the most sales?
3. How sales change overtime?
4. Which months have the highest and lowest sales
5. Which product category sells the most?
6. Evaluate customer satisfaction based on reviews
7. What recommendations can be made based on analysis?
8. Present the insights

## Tools used

1. Microsoft Excel
2. Pivot Tables
3. Charts
4. Visual studio Code
5. Kaggle
